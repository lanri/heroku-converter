HOW TO INSTALL CONVERTER APP IN MY COMPUTER



1. Download new release jar

Download Converter.jar

2. Open Command Prompt , Check java availibilty

java -version

3.a. If java 8 available , run jar

3.b. If java 8 not available , install java 8 first

https://www.java.com/en/download/manual.jsp

4. Run jar

java -jar c:\pathtojarfile.jar
